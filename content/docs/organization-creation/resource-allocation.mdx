# 자원 할당

## 1. 자원 할당 개요

org_admin은 조직에 속한 팀에 조직이 보유한 수량만큼 자원을 할당할 수 있습니다.

## 2. 자원 할당 시작

팀 리스트에서 자원을 할당하고 싶은 팀의 ① 버튼을 클릭합니다.

![자원 할당 버튼 클릭](/docs/3-4-1.svg)

## 3. 수량 선택 및 저장

팝업에서 원하는 수량을 선택한 후 **Save** ① 버튼을 클릭합니다.

![수량 선택 및 저장](/docs/3-4-2.svg)

## 4. 할당 확인

① 해당 자원이 할당되었는지 확인합니다.

![자원 할당 확인](/docs/3-4-3.svg)

<Callout type="info">
  해당 팀의 team_admin과 developer는 팀에 할당된 자원만 사용할 수 있습니다.
</Callout>
