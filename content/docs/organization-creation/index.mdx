# 조직생성

## 1. Organizations 클릭

상단 네비게이션 또는 사이드바에서 **Organizations** ①을 클릭합니다.

![Organizations 클릭](/docs/3-1-1.svg)

## 2. Convert to Organization Account 선택

**Convert to Organization Account** ①를 선택합니다.

![Convert to Organization Account](/docs/3-1-2.svg)

## 3. 조직 이름 작성 및 동의

조직 이름 ①을 작성하고 이용 약관에 동의 ②합니다.

![조직 이름 작성](/docs/3-1-3.svg)

## 4. 생성 확인

생성 후 나오는 페이지에서 리스트에 생성한 조직이 표출되는지 ① 확인하고,
프로필에 마우스를 올렸을 때 조직 계정이 생성된 것을 ② 확인합니다.

![생성 확인](/docs/3-1-4.svg)

## 5. 조직 페이지 확인

위 ②를 클릭하고 **Organizations**에 진입했을 때 아래 형식의 페이지가 표출되는지 확인합니다.

![조직 페이지](/docs/3-1-5.svg)

## 6. GPU 할당 신청

위 작업이 완료되면 연락 주시면 GPU 할당을 받으실 수 있습니다.

<Callout type="info">
  조직 생성 완료 후 담당자에게 연락하시면 GPU 자원을 할당해 드립니다.
</Callout>
