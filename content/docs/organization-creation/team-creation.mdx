# 팀 생성

## 1. Create New Team 클릭

조직 계정의 Organizations 페이지에서 **Create New Team** ① 버튼을 클릭합니다.

![Create New Team 클릭](/docs/3-2-1.svg)

## 2. 팀 이름 입력

나온 모달에서 팀 이름 ①을 기입한 후 ② 버튼을 클릭합니다.

![팀 이름 입력](/docs/3-2-2.svg)

## 3. 생성 확인

생성 후 Organization 페이지에서 다음 내용을 확인합니다.

①에 팀이 생성되었는지 확인하고, ②에 해당 팀 team_admin 초대 코드가 생성되었는지 확인합니다.
③으로 team_admin 초대 링크를 복사합니다.

![생성 확인](/docs/3-2-3.svg)

<Callout type="info">
  team_admin은 팀 생성 후 최초 3일만 등록 가능합니다.
</Callout>
