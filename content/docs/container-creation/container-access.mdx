# 컨테이너 접속

## 1. 케밥 메뉴 선택

① 우측 케밥 메뉴를 선택합니다.

![케밥 메뉴 선택](/docs/4-2-1.svg)

## 2. Connect 선택

① **Connect**를 선택합니다.

![Connect 선택](/docs/4-2-2.svg)

## 3. 접속 방식 선택

① **VS Code**를 선택한 후 ② **Connect**를 클릭합니다.

<Callout type="info">
  Web Shell은 현재 개발 중입니다.
</Callout>

![접속 방식 선택](/docs/4-2-3.svg)

## 4. 코드 서버 페이지 확인

코드 서버 페이지가 표출되는지 확인합니다.

<Callout type="info">
  세션이 만료된 경우 로그인 후 접근할 수 있습니다.
</Callout>

![코드 서버 페이지 확인](/docs/4-2-4.png)
