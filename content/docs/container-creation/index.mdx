# 컨테이너 생성

## 1. GPU 인스턴스 생성 개요

자원(GPU)이 할당되면 해당 자원으로 GPU 인스턴스를 생성할 수 있습니다.

## 2. Pod 메뉴 진입 및 생성 시작

① Pod 메뉴에 진입한 후 ② GPU 인스턴스 생성을 시작합니다.

![Pod 메뉴 진입](/docs/4-1-1.svg)

## 3. 자원(GPU) 선택

① 원하는 자원(GPU)을 선택합니다.

![자원 선택](/docs/4-1-2.svg)

## 4. 컨테이너 이름 및 요금 설정

① 컨테이너 이름을 입력하고 ② 요금 설정을 클릭합니다.

<Callout type="info">
  요금 설정 선택 여부로 현재 즉시 과금되거나 하지 않습니다.
</Callout>

![컨테이너 이름 및 요금 설정](/docs/4-1-3.svg)

## 5. 위치 선택

① 위치를 선택합니다.

<Callout type="info">
  현재는 신사만 선택 가능합니다.
</Callout>

![위치 선택](/docs/4-1-4.svg)

## 6. 저장소 설정

컨테이너에 mount할 ② 저장소를 생성하거나 ① 스킵합니다.

![저장소 설정](/docs/4-1-5.svg)

## 7. Launch Instance 클릭

생성 후 ① **Launch Instance** 버튼을 클릭합니다.

![Launch Instance 클릭](/docs/4-1-6.svg)

## 8. 약관 동의

팝업에서 ① **I agree to the above**를 클릭합니다.

![약관 동의](/docs/4-1-7.svg)

## 9. 컨테이너 생성 확인

① 컨테이너가 생성되었는지 확인합니다.

![컨테이너 생성 확인](/docs/4-1-8.svg)
