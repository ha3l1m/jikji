# 계정생성

## 1. jikjicloud.io 접속

브라우저에서 [https://jikjicloud.io](https://jikjicloud.io) 에 접속합니다.

## 2. 로그인 페이지

로그인이 되어 있지 않은 경우 아래와 같은 로그인 페이지가 표출됩니다.

![로그인 페이지](/docs/image2-2.png)

## 3. OAuth 로그인

Google, GitHub 등 4개의 OAuth 서비스 중 하나를 선택하여 로그인합니다.
아래는 Google 계정으로 진행하는 예시입니다.

### a. 계정 선택

구글 계정 선택 화면에서 사용할 계정을 선택합니다.

![구글 계정 선택](/docs/image2-3-a.png)

### b. 액세스 허용

앱 접근 권한 허용 화면에서 **허용**을 클릭합니다.

> 최초 로그인 이후에는 앱 권한을 직접 삭제하기 전까지 해당 화면이 다시 표출되지 않습니다.

![액세스 허용](/docs/image2-3-b.png)

### c. 로그인 완료

로그인 성공 시 jikjicloud.io 메인 페이지로 이동합니다.
